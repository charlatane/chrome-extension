## Run Locally

Clone the project

```bash
  git clone https://github.com/charlatane/chrome-extension
```

Go to Chrome

```bash
  go to Manage extensions > Load Unpacked > upload the folder
```

Important

```bash
make sure your "Developer's mode" is turned on
```
